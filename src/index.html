Ubic — polymorphic service manager

Key features

  • Built-in watchdog
  • Global service tree
  • Custom status checks
  • Polymorphic service classes
  • Code reuse, flexibility and polymorphism
  • Ability to dynamically populate service tree from any source
  • Cross-platform
  • Doesn't require pid 1 or root access

Learn more »

Installation

Debian/Ubuntu

Install from PPA: 
apt-add-repository ppa:berekuk/ubic
apt-get update
apt-get install ubic

Any POSIX-compatible system

Install from CPAN: 
cpan Ubic
ubic-admin setup
